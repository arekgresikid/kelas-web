## Menyiapkan VS Code & Extension
Langkah pertama bukan mengetik kode, tapi menyiapkan "meja kerja" Anda. Gunakan **VS Code** (Visual Studio Code) karena ini adalah standar industri.

> [!TIP]
> Ekstensi wajib untuk pemula:
> - **Prettier**: Merapikan kode secara otomatis saat di-save.
> - **Auto Close Tag**: Menutup tag HTML secara otomatis.

## Membangun Web dengan Bantuan AI
Di era modern, Anda tidak harus menghafal semua baris kode. Anda bisa menggunakan AI (seperti ChatGPT, Claude, atau Gemini) sebagai asisten koding Anda.

### Contoh Prompt untuk AI:
> "Buatlah struktur HTML sederhana untuk sebuah website portofolio pribadi. Sertakan header dengan navigasi, bagian Hero dengan foto dan teks sambutan, serta footer dengan link media sosial. Gunakan Tailwind CSS jika memungkinkan."

### Contoh Kode Sederhana (Hasil AI):
```html
<div class="min-h-screen bg-gray-100 flex flex-col items-center justify-center">
  <h1 class="text-4xl font-bold text-blue-600">Halo, Saya Developer Masa Depan!</h1>
  <p class="text-gray-600 mt-4">Website ini dibangun dengan bantuan AI dalam hitungan detik.</p>
</div>
```

## Mengenal Markdown (.md)
Materi yang sedang Anda baca saat ini ditulis menggunakan **Markdown**. Ini adalah format teks sederhana yang sangat populer di kalangan developer untuk menulis dokumentasi.

Hal teknis yang perlu Anda tahu:
- `#` digunakan untuk Judul Besar (H1).
- `##` digunakan untuk Judul Bagian (H2).
- `**teks**` untuk menebalkan tulisan.
- ` ``` ` digunakan untuk membungkus blok kode agar rapi.

## Instalasi Git & GitHub
Website Anda harus punya "mesin waktu" agar jika terjadi kesalahan, Anda bisa kembali ke versi sebelumnya. Itulah fungsi **Git**.
1. Download dan install Git dari `git-scm.com`.
2. Buat akun di **GitHub.com** untuk menyimpan kode Anda di internet secara gratis.
3. Sambungkan Git di komputer Anda ke GitHub menggunakan perintah `git config`.
