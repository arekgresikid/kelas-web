## Inisialisasi Repository
Buka terminal di VS Code, lalu ketik perintah sakti ini:
`git init` - Ini memberitahu Git untuk mulai mengawasi folder Anda.

## Commit: Mencatat Perubahan
Ketik:
1. `git add .` - Menandai semua file untuk diunggah.
2. `git commit -m "Initial Commit"` - Memberi catatan pada versi ini.

## Membuat Repository di GitHub
1. Login ke GitHub, klik tombol **New Repository**.
2. Beri nama yang sama dengan folder projek Anda.
3. Biarkan settingannya **Public** agar orang lain bisa melihat karya hebat Anda.

## Menghubungkan Lokal ke Awan
GitHub akan memberikan beberapa baris kode (remote add). Copy dan paste ke terminal Anda. Terakhir, ketik:
`git push -u origin main`

Boom! Kode Anda sekarang sudah ada di GitHub dan siap untuk "diterbangkan" ke server. 📦
