## Unduh dan instalasi VS Code
Visual Studio Code (VS Code) adalah text editor paling populer saat ini. Anda bisa mengunduhnya secara gratis di [code.visualstudio.com](https://code.visualstudio.com). Setelah diunduh, ikuti instruksi instalasi sesuai sistem operasi Anda (Windows, macOS, atau Linux).

## Antarmuka dasar
- **Explorer**: Tempat melihat folder dan file proyek Anda.
- **Editor**: Area utama untuk menulis kode.
- **Terminal**: Tempat menjalankan perintah (seperti `npm install`).
- **Sidebar**: Berisi ikon-ikon untuk fitur lain seperti Search, Git, dan Extensions.

## Ekstensi wajib
Agar coding lebih mudah, instal beberapa ekstensi ini:
- **Prettier**: Merapikan kode secara otomatis.
- **ESLint**: Menemukan kesalahan penulisan JavaScript.
- **GitLens**: Melihat riwayat perubahan kode per baris.
- **Live Server**: Melihat hasil website secara langsung di browser setiap kali file disimpan.

## Pintasan keyboard penting
- `Ctrl + S`: Menyimpan file.
- `Ctrl + P`: Mencari file dengan cepat.
- `Ctrl + ~`: Membuka/menutup terminal.
- `Alt + Up/Down`: Memindahkan baris kode ke atas atau bawah.
