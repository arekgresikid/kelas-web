## Unduh dan instalasi VS Code
Visual Studio Code (VS Code) adalah text editor paling populer saat ini. Anda bisa mengunduhnya secara gratis di [code.visualstudio.com](https://code.visualstudio.com). Setelah diunduh, ikuti instruksi instalasi sesuai sistem operasi Anda (Windows, macOS, atau Linux).

## Antarmuka dasar
- **Explorer**: Tempat melihat folder dan file proyek Anda.
- **Editor**: Area utama untuk menulis kode.
- **Terminal**: Tempat menjalankan perintah (seperti `npm install`).
- **Sidebar**: Berisi ikon-ikon untuk fitur lain seperti Search, Git, dan Extensions.

## Ekstensi wajib (Pro Setup)
Agar coding lebih mudah, instal beberapa ekstensi ini:
- **Prettier**: Merapikan kode secara otomatis setiap kali Anda menekan `Ctrl + S`.
- **ESLint**: Menemukan kesalahan penulisan JavaScript sebelum Anda menjalankan kodenya.
- **GitLens**: Melihat siapa yang merubah baris kode tertentu dan kapan.
- **Live Server**: Website otomatis terbuka di browser dan ter-update setiap kali file disimpan.
- **Tailwind CSS IntelliSense**: Memberikan saran (autofill) untuk class Tailwind.

## Tips Vibe Coding di VS Code
- **Auto Save**: Aktifkan fitur `File > Auto Save` agar Anda tidak perlu terus-menerus menekan `Ctrl + S`.
- **Bracket Pair Colorization**: Fitur bawaan VS Code yang memberi warna berbeda pada kurung `{}` agar kode lebih mudah dibaca.
- **Stick Scroll**: Membantu Anda tetap tahu sedang berada di fungsi mana saat men-scroll kode yang panjang.

## Pintasan keyboard (Mastering)
- `Ctrl + P`: Mencari dan membuka file dengan cepat tanpa menyentuh mouse.
- `Ctrl + Shift + P`: Membuka Command Palette (pusat segala perintah VS Code).
- `Ctrl + /`: Mengubah kode menjadi komentar (atau sebaliknya).
- `Alt + Shift + F`: Merapikan seluruh kode dalam file (Format Document).
- `Ctrl + D`: Memilih kata yang sama secara berurutan untuk diedit sekaligus.

> **Tugas**: Instal VS Code dan pasang ekstensi **Prettier** serta **Live Server** sebelum lanjut ke materi berikutnya.
