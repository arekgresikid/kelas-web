## Apa itu Cloudflare?
Cloudflare adalah layanan yang berdiri di antara pengunjung website dan server Anda. Ia bertindak sebagai pelindung sekaligus pengoptimal lalu lintas data.

## Peran Utama Cloudflare
1. **DNS Management**: Mengarahkan domain Anda ke server yang tepat dengan sangat cepat.
2. **CDN (Content Delivery Network)**: Menyimpan salinan website Anda di ratusan server di seluruh dunia. Jika pengunjung dari Jakarta membuka web Anda, Cloudflare akan mengirimkan data dari server terdekat (Singapura/Jakarta) bukan dari Amerika.
3. **Keamanan (WAF & DDoS Protection)**: Memblokir serangan hacker dan bot jahat sebelum mereka mencapai website Anda.

## SSL Gratis
Cloudflare menyediakan sertifikat SSL gratis sehingga website Anda bisa menggunakan protokol `https://` yang aman dan dipercaya oleh browser.

## Cloudflare Pages & Workers
Selain infrastruktur, Cloudflare juga menyediakan:
- **Pages**: Untuk menghosting website statis langsung dari GitHub.
- **Workers**: Untuk menjalankan kode skrip di tepi jaringan (*edge*) guna memproses data lebih cepat.
