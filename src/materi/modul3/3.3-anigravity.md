## Pengenalan AI untuk coding
Vibe Coding adalah tren baru di mana kita membangun website dengan bantuan AI sebagai asisten utama. AniGravity dirancang untuk memahami instruksi bahasa manusia dan mengubahnya menjadi barisan kode yang siap pakai.

## Cara mengakses AniGravity
Anda bisa menggunakan AniGravity langsung di lingkungan pengembangan Anda atau melalui platform yang disediakan. Selain itu, Anda juga bisa menggunakan alternatif lain seperti ChatGPT, Claude, atau GitHub Copilot untuk membantu mencari solusi coding.

## Prompt efektif untuk generate kode
Agar AI memberikan hasil yang bagus, berikan instruksi yang spesifik.
**Buruk**: "Buatkan saya website."
**Bagus**: "Buatkan saya komponen Navbar menggunakan React dan Tailwind CSS. Navbar harus memiliki logo di kiri, menu di tengah, dan tombol login di kanan. Pastikan responsif di layar HP."

## Contoh: minta AI buatkan navbar responsif
Cobalah berikan prompt di atas ke AniGravity. Anda akan mendapatkan kode HTML dan CSS yang bisa langsung Anda gunakan. AI akan menangani logika menu mobile (hamburger menu) dan styling lainnya untuk Anda.

## Etika menggunakan AI
Selalu periksa kembali kode yang dihasilkan oleh AI. Jangan hanya sekadar copy-paste. Cobalah untuk memahami logika di baliknya agar Anda tetap berkembang sebagai developer dan bisa melakukan kustomisasi di masa depan.
