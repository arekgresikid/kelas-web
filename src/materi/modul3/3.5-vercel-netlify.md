## Apa itu Vercel dan Netlify?
Keduanya adalah platform *cloud* yang dirancang untuk memudahkan developer mempublikasikan website mereka ke internet secara instan tanpa harus mengelola server sendiri (*serverless*).

## Peran Vercel
Vercel adalah pencipta framework Next.js. Ia sangat dioptimalkan untuk performa tinggi.
- **Auto-deployment**: Setiap kali Anda melakukan `git push` ke GitHub, Vercel otomatis membangun dan merilis versi terbaru website Anda.
- **Edge Functions**: Menjalankan logika server di lokasi terdekat dengan user.

## Peran Netlify
Netlify mempopulerkan istilah **JAMstack**. Ia memiliki fitur yang sangat ramah untuk website statis.
- **Form Handling**: Bisa menerima input form tanpa perlu membuat server backend.
- **Split Testing**: Memungkinkan Anda menguji dua versi website secara bersamaan untuk melihat mana yang lebih baik.

## Kapan Menggunakan Keduanya?
- Gunakan **Vercel** jika Anda membangun web dengan framework seperti Next.js atau React karena integrasinya yang sangat mulus.
- Gunakan **Netlify** untuk proyek statis sederhana atau jika Anda memerlukan fitur praktis seperti manajemen form dan autentikasi instan.
