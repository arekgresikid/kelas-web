## Menghubungkan ke GitHub
Langkah pertama adalah memastikan kode Anda sudah ada di repository GitHub. Platform seperti Vercel dan Netlify bekerja paling baik jika terhubung langsung dengan GitHub.

## Proses Deployment di Vercel
1. Login ke Vercel menggunakan akun GitHub Anda.
2. Klik **Add New** > **Project**.
3. Pilih repository yang ingin dideploy.
4. Klik **Deploy**. 
5. Tunggu sekitar 1-2 menit, dan website Anda akan aktif dengan alamat acak (misal: `proyek-saya.vercel.app`).

## Proses Deployment di Netlify
1. Login ke Netlify dengan GitHub.
2. Klik **Add new site** > **Import an existing project**.
3. Pilih GitHub dan pilih repository Anda.
4. Klik **Deploy site**.
5. Netlify akan memberikan alamat unik (misal: `nama-situs.netlify.app`).

## Keuntungan CI/CD
Dengan menghubungkan GitHub ke hosting, setiap kali Anda memperbaiki kode dan melakukan `git push`, website Anda akan terupdate secara otomatis dalam hitungan detik. Inilah yang disebut **Continuous Integration / Continuous Deployment (CI/CD)**.
