## Melakukan Build Project
Sebelum diunggah ke hosting, kode sumber Anda (React, Tailwind, dll) harus diubah menjadi file yang dimengerti browser (HTML, CSS, JS murni). Proses ini disebut **Build**. Pada Vite, kita menggunakan perintah `npm run build`.

## Optimasi Aset
Pastikan semua gambar telah dikompresi agar loading website cepat. Gunakan format modern seperti WebP atau AVIF jika memungkinkan. Website yang lambat akan ditinggalkan oleh pengunjung.

## Environment Variables
Jika website Anda menggunakan API Key atau rahasia lainnya, pastikan tidak bocor ke GitHub. Gunakan file `.env` untuk menyimpan data rahasia tersebut dan pastikan penyedia hosting Anda (Vercel/Netlify) sudah dikonfigurasi untuk membaca data tersebut.

## Final Testing
Coba buka website Anda di berbagai ukuran layar (HP, Tablet, Desktop) dan pastikan semua tombol berfungsi. Periksa juga apakah ada link yang rusak (*broken links*).
