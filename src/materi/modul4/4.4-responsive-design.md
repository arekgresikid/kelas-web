## Masalah Layar yang Berbeda
Dulu, developer membuat website hanya untuk desktop. Sekarang, lebih dari 60% orang membuka web lewat HP. Website Anda harus **Responsif**.

## Flexbox: Mengatur Baris & Kolom
Flexbox adalah cara termudah untuk mengatur posisi elemen agar sejajar atau bertumpuk.
- `display: flex` - Mengaktifkan mode flex.
- `justify-content: center` - Menaruh elemen di tengah secara horizontal.

## CSS Grid: Layout Kotak-Kotak
Grid digunakan untuk layout yang lebih kompleks (seperti layout majalah atau dashboard).
```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 kolom sama besar */
}
```

## Media Queries: Pintu Otomatis
Ini adalah teknik rahasia agar desain berubah saat layar mengecil.
```css
@media (max-width: 600px) {
  body {
    background-color: lightblue; /* Berubah hanya di HP */
  }
}
```

> [!TIP]
> Mulailah mendesain dari tampilan HP dulu (**Mobile First**), baru kemudian sesuaikan untuk layar laptop. Ini jauh lebih mudah! 📱💻
