## Cara menyematkan CSS
Ada tiga cara menambahkan CSS ke website:
1. **Inline**: Langsung di dalam tag HTML (`style="..."`).
2. **Internal**: Di dalam tag `<style>` pada bagian `<head>`.
3. **External**: Menggunakan file terpisah (misal `style.css`) dan dihubungkan menggunakan tag `<link>`.

## Selector
Selector digunakan untuk memilih elemen mana yang ingin dihias:
- **Element**: `p { color: blue; }` (semua paragraf).
- **Class**: `.konten { font-size: 16px; }` (elemen dengan class="konten").
- **ID**: `#header { background: grey; }` (hanya elemen dengan id="header").

## Box Model
Setiap elemen di web dianggap sebagai sebuah kotak. Komponennya adalah:
- **Content**: Isi dari elemen.
- **Padding**: Jarak antara isi dan garis tepi (border).
- **Border**: Garis tepi elemen.
- **Margin**: Jarak antara elemen tersebut dengan elemen lain di luarnya.

## Flexbox dan Grid
- **Flexbox**: Digunakan untuk mengatur tata letak satu dimensi (baris atau kolom). Sangat berguna untuk membuat navbar atau meratakan elemen.
- **Grid**: Digunakan untuk tata letak dua dimensi (baris dan kolom secara bersamaan). Cocok untuk struktur halaman yang kompleks.

## Responsive dengan Media Query
Media query memungkinkan kita mengubah desain web tergantung ukuran layar perangkat.
```css
@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
```
Ini berarti jika lebar layar kurang dari 600px, warna latar belakang akan berubah.
