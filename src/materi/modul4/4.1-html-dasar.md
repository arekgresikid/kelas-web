Jika Anda menganggap HTML hanyalah cara membuat teks besar atau kecil, Anda salah besar. HTML adalah tentang **Struktur dan Makna**.

## 1. Anatomi Elemen HTML
Setiap elemen HTML biasanya terdiri dari tiga bagian:
*   **Tag Pembuka**: `<tagname>`
*   **Isi (Content)**: Teks atau elemen lain di dalamnya.
*   **Tag Penutup**: `</tagname>`

Contoh: `<p class="text-blue-500">Halo Dunia</p>`
Di sini, `p` adalah nama tag, `class` adalah **Atribut** (informasi tambahan), dan "Halo Dunia" adalah isinya.

## 2. Struktur Dasar HTML5
Setiap file `.html` yang profesional harus memiliki kerangka ini:
```html
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <title>Nama Website Anda</title>
</head>
<body>
  <!-- Konten Anda di sini -->
</body>
</html>
```

## 3. Revolusi Semantic HTML (Penting!)
Di masa lalu, developer sering menggunakan `<div>` (kotak tanpa makna) untuk semua hal. Sekarang, browser dan Google ingin Anda menggunakan tag yang **bermakna**:

*   `<header>`: Untuk bagian kepala website/navigasi.
*   `<main>`: Untuk konten utama yang unik di halaman tersebut.
*   `<section>`: Untuk mengelompokkan konten yang berkaitan.
*   `<article>`: Untuk konten yang bisa berdiri sendiri (seperti blog post).
*   `<footer>`: Untuk bagian kaki website (copyright, link tambahan).

**Kenapa harus Semantik?**
1.  **SEO**: Google akan jauh lebih mudah menaikkan peringkat website Anda karena ia tahu bagian mana yang merupakan judul artikel dan mana yang hanya navigasi.
2.  **Aksesibilitas**: Teman-teman tunanetra yang menggunakan alat pembaca layar akan sangat terbantu karena alat tersebut bisa memberi tahu: "Sekarang Anda sedang berada di bagian navigasi utama".

## 4. Elemen yang Sering Digunakan
*   **Heading (`<h1>` - `<h6>`)**: Gunakan `<h1>` hanya sekali per halaman sebagai judul utama.
*   **List (`<ul>`, `<ol>`, `<li>`)**: Gunakan untuk daftar yang tidak berurutan (bullet) atau berurutan (angka).
*   **Anchor (`<a>`)**: Jembatan antar halaman menggunakan atribut `href`.
*   **Media (`<img>`, `<video>`)**: Selalu gunakan atribut `alt` pada gambar untuk menjelaskan isinya jika gagal dimuat.
*   **Form (`<input>`, `<button>`)**: Pintu masuk interaksi user untuk mengirimkan data.

## 5. HTML sebagai "Kontrak Data"
Pikirkan HTML sebagai kontrak antara Anda dan Google. 
*   Jika Anda menggunakan `<h1>`, Anda berjanji: "Ini adalah topik paling penting di halaman ini". 
*   Jika Anda menggunakan `<a>`, Anda berjanji: "Ini adalah link yang bisa diklik".
*   Google akan menghargai website yang menepati janjinya. Website dengan struktur yang jujur dan rapi akan mendapatkan peringkat lebih tinggi secara gratis.

## 6. Aksesibilitas (A11y): Standar Profesional
Developer hebat tidak hanya membuat web untuk orang normal, tapi untuk **semua orang**.
*   **Alt Text**: Selalu isi deskripsi pada gambar agar orang buta tahu gambar apa itu melalui suara dari HP mereka.
*   **Warna & Kontras**: Pastikan teks mudah dibaca (bukan abu-abu muda di atas putih).
*   **Keyboard Friendly**: Website Anda harus bisa digunakan hanya dengan menekan tombol `Tab` di keyboard tanpa bantuan mouse.

---

> **Ingat**: HTML adalah tentang **APA** kontennya, bukan **BAGAIMANA** tampilannya. Masalah tampilan (warna, posisi) adalah tugas CSS yang akan kita bahas di materi selanjutnya. 🏗️
