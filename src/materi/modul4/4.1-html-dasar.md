## Struktur dokumen HTML
Setiap file HTML harus diawali dengan kerangka dasar berikut:
```html
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <title>Judul Halaman</title>
</head>
<body>
  <h1>Halo Dunia!</h1>
</body>
</html>
```

## Elemen heading, paragraf, dan link
- **Heading**: Digunakan untuk judul (h1 sampai h6).
- **Paragraf**: Menggunakan tag `<p>` untuk membungkus teks.
- **Link**: Menggunakan tag `<a>` dengan atribut `href` untuk berpindah halaman.
- **Gambar**: Menggunakan tag `<img>` dengan atribut `src` untuk menampilkan gambar.

## List dan Tabel
- **Unordered List (`<ul>`)**: Menggunakan peluru (bullet points).
- **Ordered List (`<ol>`)**: Menggunakan angka (1, 2, 3).
- **Tabel (`<table>`)**: Digunakan untuk menampilkan data terstruktur dengan baris (`<tr>`) dan kolom (`<td>`).

## Form dan Input Element
Form digunakan untuk menerima input dari user. Contoh:
- `<input type="text">` untuk teks singkat.
- `<input type="email">` untuk alamat email.
- `<button>` untuk mengirimkan data.

## Elemen Semantik
Elemen semantik membantu mesin pencari (SEO) dan alat pembaca layar untuk memahami struktur web Anda. Gunakan tag seperti `<header>`, `<nav>`, `<main>`, `<article>`, dan `<footer>` alih-alih hanya menggunakan `<div>`.
