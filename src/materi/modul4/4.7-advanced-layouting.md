## 1. Mengapa Layouting Itu Penting?
Tanpa teknik layouting yang benar, elemen website Anda hanya akan menumpuk dari atas ke bawah. Flexbox dan Grid adalah solusi modern yang menggantikan teknik lama (seperti `float` atau `table`) yang jauh lebih sulit dikelola.

---

## 2. Flexbox: Layout Satu Dimensi
Flexbox sangat ideal untuk mengatur elemen dalam satu baris (row) atau satu kolom (column).

### Konsep Utama:
- **Flex Container**: Elemen induk (parent) yang diberi properti `display: flex`.
- **Flex Items**: Elemen anak (children) di dalamnya.

### Properti Populer:
- `justify-content`: Mengatur perataan horizontal (start, center, end, space-between).
- `align-items`: Mengatur perataan vertikal.
- `flex-direction`: Menentukan arah (row atau column).

```css
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

---

## 3. CSS Grid: Layout Dua Dimensi
Jika Flexbox untuk satu arah, Grid dirancang untuk mengatur baris DAN kolom sekaligus. Ini sangat powerful untuk membuat layout dashboard atau galeri yang kompleks.

### Konsep Utama:
- `grid-template-columns`: Menentukan jumlah dan lebar kolom.
- `grid-template-rows`: Menentukan tinggi baris.
- `gap`: Memberikan jarak antar kotak (cell).

```css
.dashboard-grid {
  display: grid;
  grid-template-columns: 200px 1fr 1fr;
  gap: 20px;
}
```

---

## 4. Kapan Menggunakan Flexbox vs Grid?
- **Gunakan Flexbox** untuk: Menu navigasi, perataan konten di dalam kotak, atau susunan tombol.
- **Gunakan Grid** untuk: Struktur halaman utama (layout besar), bento grid, atau dashboard yang membutuhkan presisi baris dan kolom.

---

## 5. Latihan Mandiri: Bento Grid
Cobalah membuat layout "Bento Grid" (seperti dashboard Apple atau GitHub) menggunakan CSS Grid. Gunakan `grid-column: span 2` untuk membuat salah satu kotak lebih lebar dari yang lain.
