## Apa itu Tailwind CSS?
Jika CSS biasa mengharuskan Anda membuat file terpisah dan memikirkan nama class (seperti `.card-container-wrapper`), **Tailwind** memungkinkan Anda menulis gaya langsung di dalam tag HTML menggunakan "Utility Classes".

## Filosofi Utility-First
Bukannya membuat satu class raksasa untuk satu komponen, Anda menggabungkan class-class kecil untuk membentuk tampilan yang diinginkan.
- `bg-black`: Warna latar hitam.
- `text-white`: Warna teks putih.
- `p-6`: Padding ukuran 6.
- `rounded-2xl`: Sudut sangat melengkung.

## Keunggulan Utama (Modern Workflow)

### 1. Desain Responsif Tanpa Pusing
Cukup tambahkan prefix untuk ukuran layar tertentu.
```html
<div class="w-full md:w-1/2 lg:w-1/3">...</div>
```
Artinya: Lebar penuh di HP, setengah layar di Tablet, sepertiga layar di Laptop.

### 2. Dark Mode Instan
Cukup tambahkan prefix `dark:`. Platform **KelasWeb** menggunakan fitur ini!
```html
<div class="bg-white dark:bg-black text-black dark:text-white">
  Konten Adaptif
</div>
```

### 3. State Management (Hover, Focus, Group)
Sangat mudah membuat interaksi tombol yang mewah.
```html
<button class="bg-blue-500 hover:bg-blue-600 hover:scale-105 transition-all">
  Klik Saya
</button>
```

## Ekstrak Komponen? (Clean Code)
Jika kodenya terlalu panjang, Anda tidak perlu takut. Dalam dunia modern (React/Next.js), kita membagi kode menjadi komponen kecil, sehingga class Tailwind yang panjang tidak menjadi masalah.

> [!TIP]
> Kuasai dasar CSS Box Model dulu, maka Tailwind akan terasa seperti memiliki "kekuatan super" dalam membangun UI yang cantik dalam waktu singkat! 🚀
