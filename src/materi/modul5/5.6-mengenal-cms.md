## Apa itu CMS?
**CMS (Content Management System)** adalah perangkat lunak yang memungkinkan pengguna non-teknis (seperti admin toko atau penulis berita) untuk mengubah konten website tanpa harus menyentuh kode program.

## Jenis-Jenis CMS:

### 1. Traditional CMS (Contoh: WordPress)
Semua jadi satu. Tampilan (Frontend) dan pengelola konten (Backend) sudah menyatu. 
- **Kelebihan**: Sangat cepat untuk membuat web blog atau company profile.
- **Kekurangan**: Terkadang berat dan sulit dikustomisasi secara ekstrem.

### 2. Headless CMS (Contoh: Contentful, Sanity, Strapi)
Ini adalah tren modern. CMS hanya berfungsi sebagai database konten yang menyediakan API. Tampilannya kita buat sendiri menggunakan React atau Next.js.
- **Kelebihan**: Sangat fleksibel dan website jadi super cepat.
- **Kekurangan**: Butuh kemampuan koding yang lebih tinggi.

## Kapan Harus Pakai CMS?
Gunakan CMS jika website Anda memiliki konten yang sering berubah (seperti berita, stok barang, atau artikel tutorial) agar Anda tidak perlu mengedit file HTML setiap kali ada perubahan. 📝
