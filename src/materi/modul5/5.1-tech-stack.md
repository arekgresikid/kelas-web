## Pengertian tech stack
Tech stack adalah kumpulan teknologi (bahasa pemrograman, framework, database, dan alat) yang digunakan untuk membangun dan menjalankan sebuah aplikasi web. Pemilihan tech stack sangat menentukan performa, skalabilitas, dan kecepatan pengembangan sebuah proyek.

## Frontend stack
Bagian yang dilihat dan digunakan langsung oleh user. Komponen utamanya adalah HTML, CSS, dan JavaScript. Saat ini, developer sering menggunakan framework/library seperti **React**, **Vue**, atau **Next.js** untuk mempercepat proses pembuatan UI.

## Backend stack
Bagian yang berjalan di server dan menangani logika bisnis, pengolahan data, serta keamanan. Contoh bahasa populer untuk backend adalah:
- **Node.js** (JavaScript)
- **Python** (Django/FastAPI)
- **PHP** (Laravel)
- **Go**

## Database stack
Tempat menyimpan data aplikasi. Ada dua jenis utama:
- **Relasional (SQL)**: Data disimpan dalam tabel terstruktur (contoh: PostgreSQL, MySQL).
- **Non-Relasional (NoSQL)**: Data disimpan dalam format dokumen yang lebih fleksibel (contoh: MongoDB).

## Contoh tech stack populer
- **MERN Stack**: MongoDB, Express, React, Node.js.
- **LAMP Stack**: Linux, Apache, MySQL, PHP.
- **JAMstack**: JavaScript, API, Markup (fokus pada website statis yang cepat).
