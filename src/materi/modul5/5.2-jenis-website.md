## Website Statis
Website statis adalah jenis website yang kontennya tetap bagi setiap pengunjung. File yang dikirimkan server sama persis dengan yang disimpan.
- **Kelebihan**: Sangat cepat, aman, dan murah (bisa hosting gratis di GitHub Pages).
- **Kekurangan**: Sulit diperbarui jika kontennya banyak, tidak ada interaksi database.

## Website Dinamis
Website dinamis bisa menampilkan konten yang berbeda-beda tergantung siapa yang membukanya, waktu, atau data terbaru.
- **Contoh**: Media sosial, situs berita, e-commerce.
- **Cara Kerja**: Menggunakan bahasa backend (seperti PHP/Node.js) untuk mengambil data dari database sebelum dikirim ke browser.

## SPA vs MPA
- **MPA (Multi Page Application)**: Browser memuat ulang seluruh halaman setiap kali kita klik menu (cara tradisional).
- **SPA (Single Page Application)**: Hanya memuat data yang diperlukan saja tanpa memuat ulang seluruh halaman, sehingga terasa sangat cepat seperti aplikasi HP (contoh: Gmail, Facebook).

## Kategori Berdasarkan Fungsi
- **Company Profile**: Menampilkan informasi perusahaan.
- **Blog**: Berisi artikel atau jurnal pribadi.
- **E-commerce**: Toko online untuk berjualan produk.
- **Portfolio**: Menampilkan karya atau resume profesional.
- **SaaS (Software as a Service)**: Aplikasi yang bisa disewa secara online.
