## 1. Apa itu Hooks?
Hooks adalah fitur yang memungkinkan Anda menggunakan "state" dan fitur React lainnya tanpa harus menulis class. Ini membuat kode lebih pendek, bersih, dan mudah dibaca.

---

## 2. useState: Mengelola Data Dinamis
`useState` digunakan untuk menyimpan data yang bisa berubah-ubah di dalam komponen. Ketika data ini berubah, React akan otomatis memperbarui tampilan (UI).

```jsx
import { useState } from 'react';

function Counter() {
  const [score, setScore] = useState(0);

  return (
    <div>
      <p>Skor Anda: {score}</p>
      <button onClick={() => setScore(score + 1)}>Tambah Skor</button>
    </div>
  );
}
```

---

## 3. useEffect: Menangani Side Effects
`useEffect` digunakan untuk menjalankan kode pada waktu tertentu, seperti saat komponen pertama kali muncul (mount) atau saat data tertentu berubah. Biasanya digunakan untuk mengambil data dari API (fetch).

```jsx
useEffect(() => {
  console.log("Halaman baru saja dibuka!");
  // Kode untuk ambil data dari server di sini
}, []); // Array kosong berarti hanya jalan sekali di awal
```

---

## 4. Pentingnya Immutability
Di React, Anda **tidak boleh** mengubah state secara langsung seperti `score = 5`. Anda harus selalu menggunakan fungsi pengubahnya (`setScore(5)`). Ini agar React tahu bahwa ada perubahan dan perlu merender ulang halaman.

---

## 5. Tips Belajar Hooks
Jangan mencoba menghafal semua Hooks sekaligus. Fokuslah menguasai `useState` dan `useEffect` terlebih dahulu, karena 90% aplikasi React dibangun menggunakan kedua Hooks ini.
