## Kenapa Butuh Framework?
Saat website semakin kompleks (banyak data, banyak interaksi), mengelola kode HTML/JS biasa akan sangat melelahkan. **Framework** membantu merapikan kode tersebut menjadi komponen-komponen kecil yang bisa digunakan ulang (Reusable).

## Konsep Utama: Component-Based
Bayangkan website adalah kumpulan kotak-kotak. Ada kotak `Header`, kotak `Sidebar`, dan kotak `Konten`. Di React, Anda membuat setiap kotak secara terpisah dan menyatukannya di satu tempat.

### Props & State
- **Props**: Data yang dikirim dari luar ke dalam komponen (seperti instruksi).
- **State**: Data internal komponen yang bisa berubah (seperti jumlah Like atau isi Input).

## Next.js: "Fullstack" Frontend
Next.js bukan sekadar React biasa. Ia adalah "supercharge" yang memberikan fitur-fitur yang tidak ada di React standar:

### 1. App Router (Struktur Modern)
Next.js menggunakan folder untuk menentukan halaman. Buat folder `kontak`, maka otomatis website Anda punya halaman `/kontak`.

**Struktur File Dasar:**
- `page.tsx`: File utama untuk tampilan halaman.
- `layout.tsx`: Kerangka halaman (seperti Header/Footer) yang tidak berubah saat berpindah halaman.
- `loading.tsx`: Tampilan sementara saat data sedang dimuat (skeleton).
- `error.tsx`: Tampilan cadangan jika terjadi kesalahan sistem.

### 2. Server vs Client Components
- **Server Components (Default)**: Kode dijalankan di server. Hasilnya dikirim ke browser sudah jadi HTML. Ini membuat web **sangat cepat**.
- **Client Components**: Digunakan hanya jika Anda butuh interaksi seperti klik tombol atau animasi (`use client`).

### 3. Data Fetching
Next.js mempermudah pengambilan data dari Database atau API eksternal dengan performa yang sangat tinggi dan aman.

## Kenapa Anda Harus Belajar Ini?
Hampir semua startup teknologi dan perusahaan besar saat ini mencari developer yang menguasai **Next.js**. Ini adalah skill paling bernilai tinggi di pasar kerja frontend saat ini.

> [!IMPORTANT]
> Next.js adalah standar emas (Gold Standard) untuk membangun web aplikasi modern. Platform **KelasWeb** ini pun dibangun menggunakan Next.js! 🏆
