## 1. Apa itu SQL?
SQL (*Structured Query Language*) adalah bahasa standar yang digunakan untuk berinteraksi dengan database relasional (seperti SQLite, MySQL, atau Cloudflare D1). Jika Database adalah gudang, maka SQL adalah robot yang bertugas mengambil, menyimpan, dan merapikan barang di gudang tersebut.

---

## 2. Perintah Dasar SQL (CRUD)
Berikut adalah perintah paling umum yang wajib dikuasai:

### SELECT (Membaca Data)
```sql
SELECT * FROM users WHERE role = 'admin';
```

### INSERT (Menambah Data)
```sql
INSERT INTO materi (title, modul) VALUES ('Belajar SQL', 8);
```

### UPDATE (Mengubah Data)
```sql
UPDATE users SET name = 'Budi Santoso' WHERE id = 1;
```

### DELETE (Menghapus Data)
```sql
DELETE FROM comments WHERE status = 'spam';
```

---

## 3. Struktur Tabel
Data dalam SQL disimpan dalam **Tabel**. Setiap tabel memiliki:
- **Row (Baris)**: Satu data lengkap (misal: satu profil user).
- **Column (Kolom)**: Atribut data (misal: Nama, Email, Tanggal Lahir).
- **Primary Key**: ID unik yang tidak boleh sama antar baris.

---

## 4. Mengapa Memakai SQL?
- **Sangat Cepat**: Bisa mencari satu data di antara jutaan baris dalam hitungan milidetik.
- **Relasional**: Bisa menghubungkan data antar tabel (misal: Menghubungkan tabel `Order` dengan tabel `Customer`).
- **Standar Industri**: Hampir semua perusahaan besar menggunakan SQL.

---

## 5. Tips Belajar
Jangan menghafal semua sintaks. Fokuslah pada bagaimana cara "bertanya" kepada database menggunakan perintah `SELECT` dan `WHERE`. Itu adalah fondasi terpenting.
