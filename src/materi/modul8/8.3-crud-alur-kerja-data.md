## Apa itu CRUD?
Hampir semua aplikasi di dunia (Instagram, Tokopedia, WhatsApp) sebenarnya hanyalah kumpulan operasi **CRUD**:
- **C**reate: Membuat data baru (Posting foto baru).
- **R**ead: Membaca/Menampilkan data (Scroll feed).
- **U**pdate: Mengubah data (Edit caption).
- **D**elete: Menghapus data (Hapus postingan).

## Contoh Nyata di Database:
Bayangkan sebuah tabel **Siswa**:
1. Anda daftar akun -> **Create** baris baru di tabel.
2. Anda login -> **Read** data untuk mencocokkan password.
3. Anda ganti foto profil -> **Update** kolom foto.
4. Anda tutup akun -> **Delete** data Anda.

## Kenapa Ini Penting?
Jika Anda sudah paham cara melakukan CRUD ke database, Anda sudah bisa membangun 90% jenis aplikasi yang ada di pasar!

> [!TIP]
> Mulailah belajar CRUD dengan satu tabel sederhana sebelum mencoba menghubungkan banyak tabel yang rumit. 📊
