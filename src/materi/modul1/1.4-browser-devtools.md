## Fungsi browser selain menampilkan web
Web browser seperti Google Chrome atau Firefox bukan hanya sekadar alat untuk membaca berita atau menonton video. Bagi seorang developer, browser adalah alat utama untuk melakukan debugging (mencari kesalahan) dan menguji desain secara langsung. Browser modern memiliki fitur canggih yang disebut **Developer Tools**.

## Membuka DevTools (F12)
Cara tercepat untuk membuka Developer Tools adalah dengan menekan tombol `F12` atau `Ctrl + Shift + I` (di Windows) dan `Cmd + Option + I` (di Mac). Anda juga bisa klik kanan di area mana pun pada halaman web dan memilih **Inspect** atau **Periksa**.

## Tab Elements: Mengubah tampilan secara langsung
Tab **Elements** memungkinkan Anda melihat kode HTML dan CSS dari website yang sedang Anda buka. Di sini, Anda bisa mencoba mengubah teks, menghapus elemen, atau mengganti warna secara *real-time*. Perubahan ini hanya bersifat sementara di browser Anda dan tidak akan merusak website aslinya. Ini adalah cara terbaik untuk bereksperimen dengan desain.

## Tab Console: Menjalankan perintah JS
Tab **Console** berfungsi sebagai terminal untuk menjalankan perintah JavaScript. Anda bisa mengetik `alert('Halo!')` atau melakukan perhitungan matematika sederhana di sini. Developer sering menggunakan Console untuk melihat pesan error jika ada skrip yang tidak berjalan sebagaimana mestinya.
