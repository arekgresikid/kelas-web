## Definisi website dan halaman web
Website adalah sekumpulan halaman web yang saling terhubung dan dapat diakses melalui internet menggunakan sebuah alamat URL. Bayangkan website sebagai sebuah buku, dan setiap halaman web adalah lembar-lembar kertas di dalamnya. Semua halaman ini biasanya disimpan di sebuah komputer khusus yang disebut server yang beroperasi 24 jam nonstop.

Halaman web sendiri dibuat menggunakan bahasa markup seperti HTML. Di era modern, website bukan lagi sekadar teks statis, melainkan platform interaktif yang memungkinkan kita bertukar informasi, berbelanja, hingga bekerja secara kolaboratif secara instan.

## Perbedaan website, web app, dan blog
Meskipun sering dianggap sama, ada perbedaan mendasar di antara ketiganya:
- **Website (Informasional)**: Biasanya bersifat searah, fokus pada penyampaian informasi. Contoh: Website profil sekolah atau profil perusahaan.
- **Web App (Aplikasi Web)**: Website yang memiliki fungsi kompleks dan interaksi dua arah yang intens. Contoh: Google Sheets, Canva, atau sistem absensi online yang sedang kita bangun.
- **Blog (Jurnal Digital)**: Jenis website yang berisi artikel atau opini yang diperbarui secara rutin. Fokusnya adalah pada konten tulisan dan interaksi komentar.

## Contoh website statis vs dinamis
- **Website Statis**: Kontennya diketik langsung di kode HTML. Untuk mengubah harga barang, developer harus membuka file kodenya dan menggantinya manual. Sangat cepat tapi sulit dikelola jika datanya ribuan.
- **Website Dinamis**: Menggunakan database untuk menyimpan data. Konten halaman berubah secara otomatis tergantung siapa yang login atau apa yang dicari. Contoh: Feed Instagram Anda berbeda dengan feed teman Anda, meskipun membuka alamat yang sama.

## Cara mengakses website
Untuk mengakses website, kita memerlukan dua komponen utama:
1. **Web Browser**: Aplikasi "penerjemah" kode (seperti Chrome, Edge, atau Safari). Ia mengambil kode HTML/CSS dari server dan menampilkannya menjadi desain yang indah.
2. **URL (Uniform Resource Locator)**: Alamat unik yang Anda ketik (misal: `https://google.com`). Browser akan mengirimkan permintaan ke DNS untuk mencari tahu di mana server tersebut berada sebelum akhirnya menampilkan halamannya kepada Anda.
