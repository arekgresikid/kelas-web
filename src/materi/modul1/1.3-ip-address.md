## Apa itu IP Address?
IP Address (Internet Protocol Address) adalah label numerik yang ditetapkan untuk setiap perangkat yang terhubung ke jaringan. 
- **IPv4**: Format paling umum, terdiri dari 4 blok angka (contoh: `172.217.194.100`).
- **IPv6**: Format baru yang lebih panjang untuk menampung lebih banyak perangkat di masa depan.

## Port: Pintu masuk aplikasi
Bayangkan IP Address sebagai alamat sebuah apartemen, maka Port adalah nomor unitnya. Sebuah server bisa menjalankan banyak layanan sekaligus. Web biasanya berjalan di **Port 80** (HTTP) atau **Port 443** (HTTPS), sedangkan database mungkin di Port 3306.

## Localhost (127.0.0.1)
Saat Anda sedang membangun website di komputer sendiri, Anda menggunakan alamat khusus yaitu `127.0.0.1` atau nama `localhost`. Ini adalah alamat yang merujuk kembali ke komputer Anda sendiri (loopback address), sehingga Anda bisa mencoba website tanpa harus terhubung ke internet.

## Cara melihat IP website
Anda bisa menggunakan terminal/command prompt untuk mengetahui IP sebuah website.
- Ketik `ping google.com`
- Atau gunakan perintah `nslookup google.com`
Terminal akan menampilkan alamat IP server yang menangani domain tersebut.
