## 1. Evolusi Hosting
Dulu kita menyewa server fisik yang lambat. Sekarang, kita menggunakan **Edge Computing**.

---

### Keuntungan Infrastruktur Modern:
1.  **Latency Rendah**: Website dimuat dari server terdekat dengan lokasi pengguna.
2.  **Auto-Scaling**: Server otomatis bertambah kuat jika trafik melonjak.
3.  **CI/CD**: Setiap kali Anda update kode di GitHub, website otomatis terupdate di internet.

---

## 2. Tools yang Wajib Diketahui
Di kelas ini, kita fokus pada alat yang paling efisien dan modern:
-   **Cloudflare Pages**: Untuk hosting file frontend.
-   **Cloudflare D1**: Database SQL yang sangat cepat dan terdistribusi.
-   **Cloudflare Workers**: Untuk menjalankan logika backend di "tepi" jaringan (Edge).

---

## 3. Protokol Keamanan Dasar
Website modern tidak hanya harus cepat, tapi juga aman:
-   **HTTPS**: Enkripsi data antara browser dan server.
-   **CORS**: Mengatur siapa saja yang boleh mengakses API Anda.
-   **Auth Bridges**: Menghubungkan identitas pihak ketiga (Google/GitHub) dengan sistem Anda secara aman.
