SEO bukan hanya soal Google. Saat Anda membagikan link website ke teman di WhatsApp, apakah muncul gambar dan judul yang bagus? Jika tidak, Anda butuh **Open Graph**.

## 1. Apa itu Open Graph (OG)?
Open Graph adalah kumpulan "Label" rahasia di dalam kode website Anda yang memberitahu Sosial Media: "Eh, kalau link ini dibagikan, pakai gambar yang ini ya, dan judulnya yang ini!"

### Contoh Meta Tags OG:
```html
<meta property="og:title" content="KelasWeb - Belajar Coding dari Nol">
<meta property="og:description" content="Kuasai pembuatan web modern dalam 30 hari.">
<meta property="og:image" content="https://kelasweb.com/poster.jpg">
```

## 2. Kenapa Ini Penting?
*   **Klik Lebih Banyak**: Link dengan gambar jauh lebih sering diklik daripada link teks polosan.
*   **Kepercayaan (Trust)**: Website yang tampilannya rapi di WhatsApp terlihat lebih profesional dan bukan website penipuan.

## 3. "Jebakan Batman" SEO (Jangan Lakukan Ini!)
Banyak pemula melakukan kesalahan ini yang membuat mereka malah di-blokir oleh Google:
1.  **Keyword Stuffing**: Memasukkan kata kunci yang sama berulang-ulang secara paksa (misal: "Jual baju murah, baju murah banget, baju murah sekali"). Google benci ini.
2.  **Membeli Backlink**: Membayar orang untuk menaruh link web Anda di sembarang tempat. Google sangat pintar mendeteksi ini.
3.  **Konten Copy-Paste**: Mengambil artikel dari web lain tanpa izin. Google akan mendeteksi konten duplikat dan menghapus web Anda dari hasil pencarian.

## 4. Checklist Terakhir Sebelum "Go Public"
- [ ] Apakah semua gambar sudah punya `alt` text?
- [ ] Apakah `title` setiap halaman unik?
- [ ] Apakah `sitemap.xml` sudah didaftarkan di Search Console?
- [ ] Apakah web terbuka kurang dari 3 detik di HP?
