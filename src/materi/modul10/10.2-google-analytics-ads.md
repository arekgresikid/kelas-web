Setelah website Anda online dan terindeks, pertanyaan besarnya adalah: **"Siapa yang berkunjung dan apa yang mereka lakukan?"**

## 1. Google Analytics 4 (GA4)
Ini adalah alat paling populer di dunia untuk melihat data statistik website secara gratis.

### Cara Memulai:
1.  Buka [Google Analytics](https://analytics.google.com/).
2.  Buat **Property** baru untuk website Anda.
3.  Pilih **Data Stream** -> **Web**. Masukkan alamat website Anda.
4.  Anda akan mendapatkan **Measurement ID** (berawalan `G-XXXXXXXX`).

### Cara Memasang di Website:
*   **Cara Manual**: Copy kode "Global Site Tag" yang diberikan Google dan paste tepat di bawah tag `<head>` di file `index.html`.
*   **Cara React**: Gunakan library `react-ga4`.
```javascript
import ReactGA from "react-ga4";
ReactGA.initialize("G-XXXXXXXX");
ReactGA.send("pageview");
```

## 2. Apa yang Harus Anda Pantau?
Jangan bingung dengan banyaknya angka. Fokuslah pada 3 hal ini:
1.  **Users**: Jumlah orang unik yang berkunjung.
2.  **Engagement Time**: Berapa lama mereka betah membaca website Anda? Jika hanya 2 detik, mungkin website Anda membosankan atau terlalu lambat.
3.  **Traffic Source**: Mereka datang dari mana? (Organic Search = Gratis dari Google, Referral = Link dari web lain, Direct = Ketik langsung alamat web).

## 3. Google Ads: "Jalan Pintas Menuju Halaman 1"
Jika SEO butuh waktu berbulan-bulan, **Google Ads** bisa membuat web Anda muncul di posisi teratas dalam hitungan jam (tapi berbayar).

### Konsep Dasar Ads untuk Pemula:
*   **Keyword Based**: Anda membayar agar web Anda muncul saat orang mengetik kata tertentu (misal: "Jasa Web Murah").
*   **PPC (Pay Per Click)**: Anda hanya bayar jika orang mengklik iklan Anda. Jika hanya melihat, gratis!
*   **Conversion Tracking**: Sangat penting! Anda harus tahu berapa orang yang akhirnya **Membeli** setelah klik iklan. Google akan memberikan potongan kode kecil untuk ditaruh di halaman "Terima Kasih" Anda.

## 4. Kapan Harus Pakai Iklan?
*   **SEO**: Untuk jangka panjang (membangun aset gratis).
*   **Ads**: Untuk jualan cepat, promo terbatas, atau mengetes apakah ide bisnis Anda laku atau tidak sebelum investasi lebih dalam di SEO.

> [!TIP]
