## Langkah 1: Amankan API Key
Setelah mendapatkan API Key dari Google AI Studio, jangan pernah menuliskannya langsung di kode.
1. Buat file `.env.local` di root folder Anda.
2. Tambahkan: `VITE_GEMINI_API_KEY=your_key_here`.

## Langkah 2: Instal Library (Jika Perlu)
Anda bisa menggunakan library resmi `@google/generative-ai` untuk mempermudah komunikasi dengan model Gemini.
```bash
npm install @google/generative-ai
```

## Langkah 3: Membuat Fungsi Helper
Buatlah fungsi sederhana untuk mengirim teks ke AI dan menerima balasannya.

```javascript
import { GoogleGenerativeAI } from "@google/generative-ai";

const genAI = new GoogleGenerativeAI(import.meta.env.VITE_GEMINI_API_KEY);

export async function tanyaAI(prompt) {
  const model = genAI.getGenerativeModel({ model: "gemini-1.5-flash" });
  const result = await model.generateContent(prompt);
  const response = await result.response;
  return response.text();
}
```

## Langkah 4: Hubungkan ke UI
Sekarang, Anda bisa memanggil fungsi `tanyaAI` saat tombol diklik dan menampilkan hasilnya di layar menggunakan `useState`.

## Contoh Ide Proyek Kecil:
1. **Penerjemah Kode**: Masukkan kode CSS biasa, AI mengubahnya jadi Tailwind.
2. **AI Writer**: Masukkan topik blog, AI membuatkan kerangka (outline) artikelnya.
3. **Pencari Bug**: Masukkan kode yang error, AI memberikan saran perbaikan.

> [!TIP]
> Mulailah dengan model `gemini-1.5-flash` karena sangat cepat dan memiliki kuota gratis yang besar untuk tahap pengembangan.
