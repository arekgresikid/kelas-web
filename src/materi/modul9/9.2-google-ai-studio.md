## Apa itu Google AI Studio?
Bayangkan Google AI Studio sebagai **Laboratorium Eksperimen**. Jika ChatGPT adalah tempat untuk mengobrol santai, maka AI Studio adalah tempat Anda "merakit" otak untuk website Anda sendiri. Di sini Anda bisa mencoba model AI tercanggih Google (Gemini) secara gratis.

## Panduan Visual: Apa Saja di Dalamnya?
Saat Anda pertama kali membuka [aistudio.google.com](https://aistudio.google.com), jangan bingung dengan banyaknya tombol. Fokuslah pada 3 area utama ini:

1. **System Instruction (Kiri)**: Ini adalah "Sifat" AI. Anda memberi tahu AI ingin menjadi siapa (contoh: "Jadilah ahli pembuat website Next.js").
2. **Chat Window (Tengah)**: Tempat Anda mengetik pertanyaan dan melihat jawaban AI.
3. **Model Settings (Kanan)**: Tempat memilih model. Pilih **Gemini 1.5 Flash** untuk kecepatan, atau **Gemini 1.5 Pro** untuk kecerdasan maksimal.

---

## Langkah Praktis untuk Pemula (Vibe Coding)

### 1. Mengajarkan "Peran" ke AI
Di kotak **System Instruction**, tempelkan teks ini:
> "Kamu adalah asisten pengembang web senior. Bantu saya membuat website yang indah menggunakan Tailwind CSS. Selalu berikan penjelasan yang mudah dimengerti pemula dan berikan kode yang bisa langsung saya tempel (copy-paste)."

### 2. Meminta Struktur Website
Di kotak chat, cobalah mengetik ini:
> "Buatlah struktur HTML dan CSS (Tailwind) untuk bagian 'Hero Section' sebuah website toko kopi. Saya ingin ada judul yang besar, foto kopi yang estetik di sebelah kanan, dan tombol 'Pesan Sekarang'."

AI akan memberikan kode lengkap yang sudah "pintar" dan sesuai keinginan Anda.

## Fitur "Ajaib" untuk Developer

### Mengunggah Folder/File
Anda bisa mengklik ikon **"+"** dan mengunggah seluruh folder proyek Anda.
- **Tanya AI**: "Di mana letak kesalahan (bug) pada folder proyek saya ini?"
- **AI akan menjawab**: AI Studio memiliki memori yang sangat besar (Context Window), ia bisa membaca ribuan baris kode Anda sekaligus dalam hitungan detik.

### Mendapatkan API Key
Jika Anda ingin "memasukkan" otak Gemini ini ke dalam website Anda sendiri:
1. Klik tombol biru **"Get API key"** di sidebar kiri.
2. Klik **"Create API key in new project"**.
3. **Simpan baik-baik!** Key ini seperti kunci rumah; siapa pun yang memilikinya bisa menggunakan kuota AI Anda.

## Kenapa Pakai Ini, Bukan ChatGPT?
- **Gratis & Powerful**: Anda bisa menggunakan model "Pro" Google secara gratis (dengan batasan tertentu).
- **Khusus Developer**: Didesain agar kode yang dihasilkan lebih rapi dan bisa langsung dihubungkan ke aplikasi.
- **Tanpa Sensor Berlebih**: Lebih fleksibel dalam membantu Anda memecahkan logika koding yang rumit.
