## React
React adalah library JavaScript buatan Meta (Facebook) yang berbasis **komponen**. Anda bisa membuat bagian-bagian kecil UI (seperti Button, Card, Navbar) dan menggabungkannya menjadi halaman yang utuh. React sangat populer karena memiliki komunitas yang sangat besar.

## Vue
Vue dikenal karena kurva pembelajarannya yang lebih landai dibanding React. Ia memiliki sintaks yang sangat bersih dan mudah dipahami oleh pemula. Vue sangat bagus untuk membangun aplikasi satu halaman (SPA) dengan performa tinggi.

## TailwindCSS
Berbeda dengan CSS tradisional, Tailwind adalah framework **Utility-First**. Anda menghias elemen dengan menuliskan class langsung di HTML (contoh: `class="bg-blue-500 text-white p-4"`). Ini membuat proses desain menjadi jauh lebih cepat dan konsisten.

## Kapan menggunakan framework?
- **Gunakan Vanilla JS**: Jika website Anda sangat sederhana (hanya informasi statis) dan ingin performa paling ringan.
- **Gunakan Framework**: Jika website Anda memiliki banyak interaksi, state yang kompleks, atau dibangun oleh tim besar agar kode lebih terstruktur dan mudah dikelola.
